\begin{frame}{Historia y origen}
\begin{itemize}
\item Es un microframework para desarrollo web en Python
\item Se basa en dos pilares: Werkzeug (WSGI), Jinja2 (plantillas)
\item Es software libre (desarrollo en github)
\end{itemize}
\end{frame}

\begin{frame}{¿Para qué es bueno Flask IMHO?}

Es bueno para:
\begin{itemize}
\item Prototipado rápido
\item Requisitos hardware muy estrictos (Raspberry Pi)
\item Encaja en la filosofía de micro servicios
\end{itemize}

\vspace{1cm}

NO es bueno para:
\begin{itemize}
\item Reemplazo de Django
\item Un proyecto de gran envergadura
\end{itemize}
\end{frame}

\begin{frame}{Filosofía}
\begin{itemize}
\item Mantener el núcleo simple pero fácilmente extensible
\item Convención sobre configuración
\item Muy buena y ordenada documentación
\end{itemize}
\end{frame}
